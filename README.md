# Ping-pong
Simple two player ping pong game made in HTML Canvas and JavaScript. With highly customizable components.
### Player controls (Can be changed)
For player 1:
```
up - w
down - s
```

For player 2:
```
up - ArrowUp
down - ArrowDown
```
